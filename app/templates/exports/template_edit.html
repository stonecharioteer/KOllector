Available Variables

Use these Jinja2 variables in your template:

Book Information
  • {% raw %}{{ book.clean_title }}{% endraw %} - Book title (enriched)
  • {% raw %}{{ book.raw_title }}{% endraw %} - Book title (from file)
  • {% raw %}{{ book.clean_authors }}{% endraw %} - Authors (enriched)
  • {% raw %}{{ book.raw_authors }}{% endraw %} - Authors (from file)
  • {% raw %}{{ book.description }}{% endraw %} - Book description
  • {% raw %}{{ book.language }}{% endraw %} - Book language
  • {% raw %}{{ book.identifiers }}{% endraw %} - ISBN/identifiers
Highlights
  • {% raw %}{% for highlight in highlights %}...{% endfor %}{% endraw %} - Loop through selected highlights
  • {% raw %}{{ highlight.text }}{% endraw %} - Highlight text
  • {% raw %}{{ highlight.chapter }}{% endraw %} - Chapter name
  • {% raw %}{{ highlight.page_number }}{% endraw %} - Page number
  • {% raw %}{{ highlight.datetime }}{% endraw %} - Highlight timestamp
  • {% raw %}{{ highlight.color }}{% endraw %} - Highlight color
  • {% raw %}{{ loop.index }}{% endraw %} - Current highlight number (inside loop)
Reading Information
  • {% raw %}{{ read_start }}{% endraw %} - First highlight date (YYYY-MM-DD)
  • {% raw %}{{ read_end }}{% endraw %} - Last highlight date (YYYY-MM-DD)
Export Metadata
  • {% raw %}{{ current_date }}{% endraw %} - Export date (YYYY-MM-DD)
  • {% raw %}{{ current_timestamp }}{% endraw %} - Export timestamp (YYYY-MM-DD HH:MM:SS)
Cover Image

The book cover is automatically included in the zip as cover.jpg or cover.png (if available). Reference it in your template:

  • Markdown: ![Cover](cover.jpg)
  • HTML: <img src="cover.jpg" alt="Cover">
Jinja2 Control Structures
  • {% raw %}{% if variable %}...{% endif %}{% endraw %} - Conditional
  • {% raw %}{% for item in items %}...{% endfor %}{% endraw %} - Loop
  • {% raw %}{{ variable or 'default' }}{% endraw %} - Default value
Tip: The default template uses Markdown frontmatter for blog compatibility. Customize it for your blog's format!
